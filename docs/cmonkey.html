NAME

Bio::KBase::cmonkey::Client

DESCRIPTION

Module KBaseCmonkey version 1.0 This module provides a set of methods for work with cMonkey biclustering tool.

Data types summary Input data types: ExpressionSeries from Expression service. ExpressionSample from Expression service. KBaseGenomes.Genome Networks.InterationSet with operons data Networks.InterationSet with STRING data Output data types: CmonkeyRun data type represents data generated by a single run of cMonkey (run_infos table of cMonkey results) CmonkeyNetwork data type represents bicluster network CmonkeyCluter data type represents a single bicluster from cMonkey network, with links to genes, experimental conditions and motifs CmonkeyMotif data type represents a single motif identifed for a bicluster

Methods summary build_cmonkey_network_job_from_ws - Starts cMonkey server run for a series of expression data stored in workspace and returns job ID of the run

build_cmonkey_network_job_from_ws

  $cmonkey_run_result_job_id = $obj->build_cmonkey_network_job_from_ws($ws_id, $params)
Parameter and return types
$ws_id is a string
$params is a Cmonkey.CmonkeyRunParameters
$cmonkey_run_result_job_id is a string
CmonkeyRunParameters is a reference to a hash where the following keys are defined:
	series_ref has a value which is a Cmonkey.expression_series_ws_ref
	genome_ref has a value which is a Cmonkey.genome_ws_ref
	operome_ref has a value which is a Cmonkey.operons_ws_ref
	network_ref has a value which is a Cmonkey.network_ws_ref
	networks_scoring has a value which is an int
	motifs_scoring has a value which is an int
expression_series_ws_ref is a string
genome_ws_ref is a string
operons_ws_ref is a string
network_ws_ref is a string
